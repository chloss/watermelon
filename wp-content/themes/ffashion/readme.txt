== Description ==

fFashion is a fully Responsive Fashion WordPress Theme suitable for a fashion blog,  fashion models website, 
a site for a  fashion clothes, fashion fashion, etc. Features include Social icons in header: Facebook,
Google+, Twitter, LinkedIn, Instagram, RSS Feeds, Tumblr, YouTube, Pinterest, Flickr, Vine, Multi-Level
Drop-Down Header Menu, Admin Customize Options (Social Sites, Slider Images and Content, Header Logo and
Footer Copyright Text), Multiple Browser Compatible (IE9, IE10, IE11, Chrome, Firefox, Opera, and Safari),
FontAwesome Integration, Full-Width Page Template, Widget Areas: Sidebar Widget Area, Homepage Widget Area,
W3C Markup Validated, Search Engine Optimized (SEO), Mobile-Friendly, and Translation Ready.

== Installation ==

Manual installation:

1. Upload the theme folder to the /wp-content/themes/ directory

Installation using 'Add New Theme'

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for theme name
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options
